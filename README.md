《游戏王》卡片查询器（跨平台版）
=============================

**本软件仅包含『中文』语言，而不是与《游戏王》本身一致，拥有各国的语言支持。**

- - -

跟随『中国OCG工作室』更新

- - -

编译方法：

### Android 原生

```
$ cd android
$ gradle build
```

### iOS 原生

```
$ cd ios
$ xcodebuild -workspace YuGiOhCard.xcworkspace/ -scheme YuGiOhCard -configuration Release
```

### 快应用版本

```
$ cd quickapp
$ npm run release
```

### Flutter 版本

```
$ cd flutter
$ flutter run
```

### PC 版本

```
准备中...
```

### 服务端(API提供)

